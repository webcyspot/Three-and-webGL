Overview

TL;DR three.js supports most modern browsers, including Internet Explorer 11 and above.

three.js provides WebGL, Canvas, SVG and CSS renderers. Different renderers may have different requirements of compatibility. In theory, three.js works on all browsers that support WebGL: Google Chrome 9+, Firefox 4+, Opera 15+, Safari 5.1+, Internet Explorer 11 and Microsoft Edge. More infomation can be found at Can I use WebGL?.

Internet Explorer Compatibily

It is strongly recommended to upgrade older versions of IE to IE11. If you still need to support IE9, you can use CanvasRenderer instead. Note that CanvasRenderer only supports a small subset of WebGLRenderer's features.

For Internet Explorer, the table below lists avaiable renderers.

Renderer IE Version
WebGLRenderer IE11+
CanvasRenderer IE9+
SVGRenderer IE9+
CSS3DRenderer IE10+
CSS2DRenderer IE9+

Polyfills

Just import polyfills base on your requirements. Taking IE9 as an example, you need to polyfill at least these features:

  • Typed Arrays
  • Blob
  • performance
  • requestAnimationFrame

Suggested polyfills

  • core-js
  • typedarray.js
  • ES6-Promise
  • Blob.js
  • performance
  • requestAnimationFrame

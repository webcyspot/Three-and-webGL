[name]

此类包含定义指数雾化的参数，即雾的浓度会随着距离指数增长

构造器

[name]( [param:Integer color], [param:Float density] )

颜色参数会传入[page:Color]构造器以设置颜色属性. 颜色可以是十六进制的或者CSS风格的字符串

属性

[property:String name]

对象的可选名称(不需要唯一)。 默认是一个空串

[property:Color color]

雾的颜色。例如:如果是黑色, 远处的对象将会被渲染为黑色

[property:Float density]

定义雾的浓度增长多快

默认是0.00025.

方法

[method:FogExp2 clone]()

返回一个新的FogExp2实例，他拥有和传入实例一样的参数

[method:FogExp2 toJSON]()

返回JSON格式的指数雾化数据

[name]

此类包含定义线性雾化的参数，即雾的浓度会随着距离线性增长

构造器

[name]( [param:Integer color], [param:Float near], [param:Float far] )

颜色参数会传入[page:Color]构造器以设置颜色属性. 颜色可以是十六进制的或者CSS风格的字符串

属性

[property:String name]

对象的可选名称(不需要唯一)。 默认是一个空串

[property:Color color]

雾的颜色。 例如:如果是黑色, 远处的对象将会被渲染为黑色

[property:Float near]

雾化开始运用的最小距离。到相机的距离小于'near'个单位的对象将不受雾化影响

默认是1

[property:Float far]

停止计算和应用雾化的最大距离。到相机的距离大于'far'个单位的对象将不受雾化影响

默认是1000

方法

[method:Fog clone]()

返回一个新的雾化实例，它有着和传入雾化实例一样的参数

[method:Fog toJSON]()

返回JSON格式的雾化

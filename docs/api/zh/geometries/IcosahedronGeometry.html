二十面几何体（[name]）

一个用于生成二十面体的类。

构造器

[name]([param:Float radius], [param:Integer detail])

radius — 二十面体的半径，默认为1。
detail — 默认值为0。将这个值设为一个大于0的数将会为它增加一些顶点，使其不再是一个二十面体。当这个值大于1的时候，实际上它将变成一个球体。

属性

[property:Object parameters]

构造函数中所包含着的每一个参数的对象。在实例化之后，该属性的任何更改都不会改变这个几何体。

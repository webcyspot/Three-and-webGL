Box3辅助工具（[name]）

一个能使得[page:Box3]能够被看到的辅助工具。

示例

var box = new THREE.Box3(); box.setFromCenterAndSize( new THREE.Vector3( 1, 1, 1 ), new THREE.Vector3( 2, 1, 3 ) ); var helper = new THREE.Box3Helper( box, 0xffff00 ); scene.add( helper );

构造器

[name]( [param:Box3 box], [param:Color color] )

[page:Box3 box] -- 将要显示的Box3对象。
[page:Color color] -- （可选） box的颜色，默认值为0xffff00。

创建一个用于表示所传入的Box3的线框盒子。

属性

请参阅其基类[page:LineSegments]来查看共有属性。

[property:Box3 box]

将要被显示的Box3。

方法

请参阅其基类[page:LineSegments]来查看共有方法。

[method:void updateMatrixWorld]( [param:Boolean force] )

它覆盖了其基类[page:Object3D]中的方法，所以 它还将更新线框盒子到[page:Box3Helper.box .box] 属性的范围。

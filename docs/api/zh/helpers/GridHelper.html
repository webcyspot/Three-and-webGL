网格辅助工具（[name]）

GridHelper是一个用于定义网格的对象，网格是由线组成的二维数组。.

示例

var size = 10; var divisions = 10; var gridHelper = new THREE.GridHelper( size, divisions ); scene.add( gridHelper ); [example:webgl_helpers Example using various helpers]

构造器

[name]( [param:number size], [param:Number divisions], [param:Color colorCenterLine], [param:Color colorGrid] )

size -- 网格的尺寸，默认值为10。
divisions -- 网格的划分数，默认值为10。
colorCenterLine -- 网格中线的颜色。这个值可以是[page:Color]对象、十六进制值或CSS颜色名。默认值为0x444444。
colorGrid -- 网格线的颜色。这个值可以是[page:Color]对象、十六进制值或CSS颜色名。默认值为0x888888。

创建一个新的每条边具有'size'大小、被分为'divisions'段的[name]。其中的颜色值是可选的。

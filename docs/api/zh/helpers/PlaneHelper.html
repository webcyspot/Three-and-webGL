平面辅助工具[name]

用于显示[page:Plane]（平面）的辅助工具。

示例

var plane = new THREE.Plane( new THREE.Vector3( 1, 1, 0.2 ), 3 ); var helper = new THREE.PlaneHelper( plane, 1, 0xffff00 ); scene.add( helper );

构造器

[name]( [param:Plane plane], [param:Float size], [param:Color hex] )

[page:Plane plane] -- 将会被可视化的平面
[page:Float size] -- （可选） 平面辅助工具的边长，默认值为1。
[page:Color color] -- （可选）平面辅助工具的颜色，默认值为0xffff00。

创建一个新的线框，以用于表示所传入的平面。 Creates a new wireframe representation of the passed plane.

属性

请参阅其基类[page:LineSegments]来查看共有属性。

[property:Plane plane]

将会被可视化的[page:Plane plane]（平面）。

[property:Float size]

plane helper（平面辅助工具）的边长。

方法

请参阅其基类[page:LineSegments]来查看共有属性。

[method:void updateMatrixWorld]( [param:Boolean force] )

该方法覆盖了其基类[page:Object3D]中的方法，因此它也将 根据[page:PlaneHelper.plane .plane]和[page:PlaneHelper.size .size]属性 来更新辅助工具对象。

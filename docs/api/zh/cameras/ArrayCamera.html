[name]

可以使用[name]以使用预定义的一组摄像机有效地渲染场景。这是渲染VR场景的重要性能方面。
[name]的实例总是有一组子摄像机。必须为每个子摄像机定义bounds属性，该属性确定使用此摄像机渲染的视口部分。

例子

[example:webgl_camera_array camera / array ]

构造函数

[name]( [param:Array array] )

摄像机组。

属性

请参阅基类[page:PerspectiveCamera]的公共属性。

摄像机组。

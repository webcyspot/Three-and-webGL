凸包缓冲几何体（[name]）

[name] 可被用于为传入的一组点生成凸包。 该任务的平均时间复杂度被认为是O(nlog(n))。

示例

[example:webgl_geometry_convex geometry / convex ]

var geometry = new THREE.ConvexBufferGeometry( points ); var material = new THREE.MeshBasicMaterial( {color: 0x00ff00} ); var mesh = new THREE.Mesh( geometry, material ); scene.add( mesh );

构造函数

[name]( [param:Array points] )

points — 所得凸包中将包含的一组[page:Vector3 Vector3]。

贴花几何体（[name]）

[name] 可被用于创建贴花网格物体，以达到不同的目的，例如：为模型增加独特的细节、进行动态的视觉环境改变或覆盖接缝。

示例

[example:webgl_decals decals ]

var geometry = new THREE.DecalGeometry( mesh, position, orientation, size ); var material = new THREE.MeshBasicMaterial( { color: 0x00ff00 } ); var mesh = new THREE.Mesh( geometry, material ); scene.add( mesh );

构造函数

[name]( [param:Mesh mesh], [param:Vector3 position], [param:Euler orientation], [param:Vector3 size] )

mesh — 一个网格对象。
position — 贴花投影器的位置。
orientation — 贴花投影器的朝向。
size — 贴花投影器的尺寸。
